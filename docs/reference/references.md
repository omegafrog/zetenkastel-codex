# 08-References

## 목적
참고한 원본 자료와 출처 정보를 보관합니다.

## 포함 내용
- 논문 PDF
- 전자책
- 강의 자료
- 웹 아티클 아카이브
- 연구 자료
- 외부 문서

## 조직 방법

### 옵션 1: 타입별
```

```

### 옵션 2: 주제별

## 메타데이터 기록
각 참고 자료에 대해 메타데이터 노트 생성:

```markdown
# [자료명]

**타입**: 책/논문/아티클/강의
**저자**: [이름]
**발행일**: YYYY-MM-DD
**URL**: [링크]
**파일**: [[첨부파일]]

## 요약
[간단한 요약]

## 관련 노트
- [[02-literature-notes/이자료]] - 내가 작성한 요약
- [[03-permanent-notes/개념]] - 이 자료에서 추출한 개념

## 태그
#책 #주제 #연도
```

## 베스트 프랙티스
- 파일명: `저자-제목-연도.pdf`
- 메타데이터 노트는 같은 이름으로: `저자-제목-연도.md`
- 웹 아티클은 PDF로 저장 (웹페이지 변경/삭제 대비)
- 저작권 주의

## 링크 패턴
Literature Notes에서 이곳으로 링크:
```markdown
**출처**: [[08-references/저자-책제목-2024]]
```
