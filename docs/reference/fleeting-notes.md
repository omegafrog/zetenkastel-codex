# 01-Fleeting Notes

## 목적
빠르게 떠오른 생각, 아이디어, 영감을 임시로 기록하는 공간입니다.

## 제텔카스텐에서의 역할
- 지식 창출의 첫 단계
- 즉흥적이고 가공되지 않은 생각
- Permanent notes의 씨앗

## 특징
- **즉흥성**: 순간적으로 떠오른 생각을 즉시 캡처
- **미완성**: 완성도가 낮아도 괜찮음
- **임시성**: 24-48시간 내 처리 권장
- **원자성 불필요**: 여러 아이디어가 섞여도 OK

## 언제 사용하는가?
- 회의 중 떠오른 아이디어
- 책/아티클 읽다가 생긴 질문
- 산책하다 떠오른 통찰
- 대화 중 메모
- 꿈에서 본 아이디어
- "나중에 생각해볼 것" 목록

## 작성 방법
1. 템플릿 사용: `09-templates/template-fleeting-note.md`
2. 빠르게 작성 (완벽함보다 속도)
3. 나중에 이해할 수 있을 정도의 맥락만 포함
4. 태그 추가: `#fleeting`

## 처리 프로세스
```
Fleeting Note 생성
    ↓
24-48시간 내 검토
    ↓
    ├─→ Permanent Note로 발전 (03-permanent-notes/)
    ├─→ 프로젝트에 추가 (04-projects/)
    ├─→ Literature Note와 연결 (02-literature-notes/)
    └─→ 삭제 (가치 없음)
```

## 네이밍 규칙
- **방법 1**: `YYYYMMDD-HHMM-주제.md`
  - 예: `20251010-1430-API설계아이디어.md`
- **방법 2**: `자연스러운 제목.md`
  - 예: `캐시 무효화 전략 아이디어.md`

## 주의사항
❌ 하지 말아야 할 것:
- Fleeting notes를 영구 보관하기
- 너무 많은 시간 투자하기
- 완벽하게 작성하려고 하기

✅ 해야 할 것:
- 정기적으로 검토하기
- 가치 있는 것만 발전시키기
- 대부분은 삭제하기 (괜찮음!)

## 성공 지표
- 주 1-2회 정기 검토
- Fleeting notes의 10-20%만 permanent notes로 발전
- 폴더에 오래된 노트(1주 이상)가 없음

## 관련 도구
- 모바일에서 빠른 캡처: Obsidian Mobile
- 음성 메모 → 텍스트 변환
- 브라우저 클리퍼

---
**핵심**: 완벽함보다 캡처, 대부분은 버려도 OK
