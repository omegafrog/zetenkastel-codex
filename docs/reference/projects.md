# 04-Projects (PARA 통합)

## 목적
마감일이 있고 명확한 목표가 있는 단기 프로젝트를 관리합니다.

## PARA의 Projects
- 시작과 끝이 명확한 작업
- 구체적인 결과물 목표
- 완료 시 Archives로 이동

## 제텔카스텐과의 통합
프로젝트 진행 중:
- **Fleeting notes** (`01-fleeting-notes/`) - 프로젝트 중 떠오른 아이디어

- **Permanent notes** (`03-permanent-notes/`) - 프로젝트에서 얻은 영구 지식
- **Project notes** (여기) - 프로젝트 실행 문서

## 프로젝트 예시
- 새 기능 개발
- 학습 목표 (예: "Spring Security 완전 정복")
- 글쓰기 프로젝트
- 리팩토링 작업
